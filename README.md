## 📌 Project Overview

DocuSmart is a **web-based platform powered by Natural Language Processing (NLP)** designed to automate the analysis, understanding, and transformation of textual documents.

The project focuses on **helping educators and professionals** reduce manual workload by leveraging AI for tasks such as document analysis, question generation, translation, and accessibility enhancement.

---

## 💡 Why This Project Matters

In today's educational and professional environments, a significant amount of time is spent on repetitive tasks such as:

- Creating educational content  
- Analyzing documents  
- Adapting materials for different audiences  
- Translating and simplifying information  

DocuSmart addresses this problem by:

- 🤖 Automating complex text analysis using AI  
- 📚 Helping educators save time and improve productivity  
- 🌍 Making content accessible in multiple languages  
- ♿ Promoting inclusive education through accessibility features  

This project demonstrates how **AI can be applied to solve real-world problems**, improving efficiency and accessibility in a meaningful way.

---

## 🧹 1. Text Processing & NLP Analysis

### 🔍 Core Features

- **Text statistics**
  - Word count, sentence count, character count

- **Automatic Question Generation**
  - AI-generated open-ended questions from text content

- **Sentiment Analysis**
  - Classification: Positive / Negative / Neutral  
  - Powered by Azure Cognitive Services

- **Readability Analysis**
  - Flesch Reading Ease score
  - Estimated educational level

- **Text Cohesion Evaluation**
  - Measures semantic connection between sentences  

- **Grammar & Spell Checking**
  - Using `language_tool_python`

- **Keyword Extraction**
  - Azure Text Analytics

- **Word Cloud Visualization**
  - Highlights most relevant terms

- **Smart Link Generation**
  - Auto-generated Google search links from keywords
